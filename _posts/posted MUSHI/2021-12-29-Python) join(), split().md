# join()

튜플, 리스트, stirng 등 반복 가능한 iterable 객체를 받는 메서드입니다.

**각각의 원소를 모아 하나의 문자열로 합쳐줍니다.**

"요소 연결시 추가할 문자".join(iterable객체)



```python
#- join()
alphabet = ['a', 'b', 'c']
",".join(alphabet)
```



실행 결과 

> 'a,b,c'
